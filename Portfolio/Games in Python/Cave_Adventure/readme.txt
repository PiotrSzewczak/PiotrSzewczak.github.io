README - Cave Adventure (EN)
Game Description
Cave Adventure is an engaging game where you pilot a spaceship through underground caves, skillfully navigating to avoid colliding with cave walls. Drawing inspiration from the popular game Flappy Bird, it offers a similar challenge and gameplay mechanics.

Controls
Upon launching the game, you can choose to control the spaceship using either the keyboard arrows or mouse buttons. To start the game, open the main.py file and press the space bar.

Features
Obstacle Randomization: The game features a unique obstacle randomization mechanic, varying the size and placement of walls, ensuring each round is diverse and unpredictable.
Scoring: At the end of each round, your score is displayed, reflecting your performance.
Requirements
The game requires the pygame library to run. Ensure it is installed in your environment.

Author
The game was developed by Piotr Szewczak.




README - Cave Adventure (PL)

Opis gry
Cave Adventure to dynamiczna gra polegająca na sterowaniu statkiem kosmicznym w podziemnych jaskiniach. Gracz musi umiejętnie manewrować statkiem, aby unikać kolizji ze ścianami jaskiń. Gra czerpie inspirację z popularnej gry Flappy Bird, oferując podobne wyzwanie i mechanikę.

Sterowanie
Po uruchomieniu gry użytkownik ma możliwość wyboru sterowania za pomocą strzałek klawiatury lub przycisków myszy. Aby rozpocząć grę, należy otworzyć plik main.py i nacisnąć spację.

Funkcje
Randomizacja przeszkód: Gra oferuje unikalną funkcję randomizacji wielkości i rozmieszczenia ścian, co zapewnia różnorodność i nieprzewidywalność każdej rundy.
Punktacja: Po zakończeniu każdej rundy gracz otrzymuje informację o zdobytych punktach.
Wymagania
Do uruchomienia gry wymagana jest biblioteka pygame. Upewnij się, że jest zainstalowana w Twoim środowisku.

Autor
Gra została stworzona przez Piotra Szewczaka.
